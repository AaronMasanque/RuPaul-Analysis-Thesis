# RuPaul-Analysis-Thesis
My Master's thesis on using machine learning to predict the season 13 winner of RuPauls Drag Race.
# Files
allrank.csv - ranking data per episode per season
contestant_data.csv - demographic information about each contestant
ranking.csv - reformatted ranking data showing the number of times a contestant landed on each type of rank
RPDR_Data.xlx - combined data
